Work

I am currently working as a freelance web developer and studying to increase my knowledge of JavaScript and PHP.

I have created a websites for various small businesses. I actively manage the hositing of sites for my clients as well as helping them with search engine optimiztion (SEO). Checkout my resume page for a list of current projects. If you have questions regarding a site you'd like to build feel free to contact me.

Typically I use the Twitter Bootstrap CSS/JavaScript framework as a starting poing for my project. The great thing about Bootstrap is it's mobile first, so there's no need to 'optimize the site for mobile.' Many web designers charge extra to create a mobile site, but because it's baked right into the frontend framework there's no extra work necessary. As a result there are no extra charges passed down to the client.

Lately I have been researching web application development, and have been working actively with AngularJS and jQuery. In addition I have been experimenting with various development work flows in node with Grunt, Bower, and Yeoman.
